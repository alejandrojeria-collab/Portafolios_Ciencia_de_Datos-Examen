Este repositorio corresponde a un proyecto de análisis y predicción del Producto Interno Bruto (PIB) de distintos países, utilizando indicadores económicos, sociales y demográficos obtenidos desde el Banco Mundial.

El trabajo se desarrolla por etapas, utilizando Python (Google Colab) para el análisis y GitHub para controlar las versiones. Cada etapa del proyecto se gestiona en una rama independiente, mientras que la rama principal mantiene la versión consolidada del trabajo.# Portafolios_Ciencia_de_Datos-Examen
